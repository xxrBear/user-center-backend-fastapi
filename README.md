![Static Badge](https://img.shields.io/badge/build-_python_3.11-blue)

## 简介
基于编程导航 `鱼皮用户中心` 后端项目，使用 FastAPI 框架重写。

## 快速部署
```shell
docker build -t user-center .
docker run -d -p 8080:8080 user-center
```
